**Project: Modern Linktree Clone - Dark Theme UI Design**

Design a modern, sleek dark-themed UI for a Linktree clone application. This is a link-in-bio platform where users create profiles and share multiple links.

**Tech Stack:**
- Next.js 16 (React)
- Tailwind CSS
- Framer Motion (animations)
- Dark theme with CSS variables

**Color Palette (Dark Theme - Purple & Dark Gray):**
- Background: #0a0a0f (primary), #141420 (secondary), #1a1a2e (tertiary)
- Cards: #1f1f35 with borders #2d2d44
- Text: #f8f9fa (primary), #cbd5e1 (secondary), #94a3b8 (muted)
- Accent: #8b5cf6 (purple), hover: #a78bfa
- Success: #10b981, Error: #ef4444

**Key Pages to Design:**

1. **Homepage/Landing:**
   - Hero section with gradient dark background
   - "Create Your Link Page" headline
   - Two CTA buttons: "Get Started" (primary purple) and "Sign In" (outlined)
   - Minimal, centered layout
   - Smooth animations on load

2. **Login/Signup Pages:**
   - Centered card design with dark background
   - Form inputs with dark theme (dark input fields, purple focus states)
   - Gradient background
   - Error messages in red
   - Animated form elements

3. **Dashboard:**
   - Top navigation bar (dark card style)
   - Two-column grid layout
   - Profile card (left): username, display name, bio, public URL preview
   - Links card (right): link count, top 3 links, manage button
   - Quick stats section: Total Links, Total Clicks, Avg Clicks (with animated numbers)
   - Cards have hover effects (slight lift + scale)
   - All cards use dark theme with subtle borders

4. **Profile Management Page:**
   - Form with dark inputs
   - Username field with availability check
   - Display name, bio textarea
   - Font selector dropdown
   - Save/Cancel buttons
   - Real-time validation feedback

5. **Link Management Page:**
   - List of links with drag handles (â˜° icon)
   - Each link shows: title, URL, description, click count
   - Edit/Delete buttons per link
   - "Add Link" button (top right)
   - Modal for add/edit with dark theme
   - Drag-and-drop reordering with visual feedback

6. **Public Profile Page:**
   - Centered layout, max-width container
   - Profile picture (circular, 120px) with border
   - Display name (large, bold)
   - Bio text
   - List of link buttons (full-width, rounded, with hover effects)
   - Customizable colors (user can set background, text, button colors)
   - Smooth entrance animations (staggered)

**Design Principles:**
- Dark theme throughout (no light backgrounds)
- Purple accent color for primary actions
- Smooth animations and micro-interactions
- Card-based layouts with subtle shadows
- Rounded corners (8-12px border-radius)
- Generous spacing and padding
- Modern typography (clean, readable)
- Glassmorphism effects on cards (subtle)
- Gradient backgrounds for depth

**Component Specifications:**
- Buttons: Rounded, purple background, white text, hover scale effect
- Inputs: Dark background (#141420), border (#2d2d44), purple focus ring
- Cards: Dark background (#1f1f35), border (#2d2d44), shadow, hover lift
- Links: Full-width buttons, rounded, with icon support, hover scale
- Navigation: Dark bar, minimal, with user email and sign out

**Animation Requirements:**
- Page transitions: Fade in (0.3s)
- Card entrance: Staggered, spring animation
- Button hover: Scale 1.05, smooth transition
- Input focus: Scale 1.02, border color change
- Loading states: Spinning loader with purple accent

**Accessibility:**
- High contrast text (WCAG AA)
- Focus states clearly visible
- Touch targets minimum 44x44px
- Keyboard navigation support

**Mobile Responsive:**
- Single column on mobile
- Stacked cards
- Full-width buttons
- Touch-friendly spacing

Create a comprehensive design system with all these pages, maintaining consistency in dark theme, purple accents, and modern aesthetics. Focus on creating a premium, polished look with smooth animations and excellent UX.
